Welcome to my project page. Here, you can find my all the latest and past projects listed with the link to their codes.

Kinodynamic RRT

In this project, we developed a kinodynamic planner for the RRT algorithms based on ROS. It was tested and visualized in rviz. The algorithm has improved time complexity, i.e., it takes less time to trace the shortest collision-free path. We have successfully deployed this algorithm by modifying it for the UR5 robotic arm. Please click here to find the algorithm in action.

RL Algorithms

This is collection of all the RL-algorithms. The project aims at developing an easy to use library for reinforcement learning algorithms like SARSA, Q-learning, etc.

python-RRT

Developed as an individual project, it is collection of all the motion planning algorithms written purely written in python like RRT, PRM, etc. It also has the source code for the program we developed at HTIC, IIT-Madras for acquiring data like joint angle, joint velocitites, etc from the UR5 robot. Some basic coding exercises based on ROS are also included in this repository.

OpenAI-Gym

This is one of my favorite projects to work. Of course, this is not my repository but an opensource project by OpenAI. Basically, it is an environment that helps in rendering proper functionality to reinforcement learning algorithms like SARSA, Monte Carlo, Q-learning. Recently, I have made my first commit to this repository by adding the Gridworld environment. Gridworld

Py-robotics: A minimalistic python library

Created a light weight python module that has robotic algorithms implemented with reinforcement learning concepts.

RTDE: Data Acquisition from UR5 robotic arm

This project is about acquiring data like joint velocities, joint angles, controller configuration, robot state, etc from the UR5 robot. We developed a program based on Real-time Data Exchange(RTDE) interface for wireless data acquisition from the robot.
